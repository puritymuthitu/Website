TEACHING MATHEMATICS USING TECHNOLOGY

Technology, when used appropriately, can be a motivator for students. When they're using technology, students can investigate and explore and try to better understang what's going on in regard to mathematics because they have the tools to help support that exploration (Hollebrands, 2020)

    GeoGebra Geogebra is a dynamic mathematics software for all levels of education that brings together geometry, algebra, spreadsheets, graphing, statistics and calculus in a easy to use package. It allows learners to visualize and manupulate hence enhancing a deeper understanding of the concept beign taught. Learners can easily install GeoGebra App using any type of a computer GeoGebra
    Desmos Desmos is an advanced graphing calculator implemented as web application and mobile application. Desmos is quick and free to set up and run simple activities. I love it because it enables instructors to monitor student progress easily Desmos
    Geometer's Sketchpad Geometer's Sketchpad is a software for teaching mathematics. It helps students visualize geometric shapes in way that is not possible on paper tools like CODP, hence increasing their engagement, understanding and achievement. Geometer's Sketchpad
    Polypad Polypad is a digital technology tool used for exploring mathematics. Good thing is as a mathematics teacher, you can use the tool to track my students' progress Polypad
