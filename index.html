My Favorite Edtech Tools

  • GeoGebra - Geogebra is a dynamic mathematics software for all levels of education that brings together geometry, algebra, spreadsheets, graphing, statistics and calculus in a easy to use package. It allows learners to visualize and manupulate hence enhancing a deeper understanding of the concept beign taught. Learners can easily install GeoGebra App using any type of a computer.
  • Desmos - Desmos is an advanced graphing calculator implemented as web application and mobile application. Desmos is quick and free to set up and run simple activities. I love it because it enables instructors to monitor student progress easily.
